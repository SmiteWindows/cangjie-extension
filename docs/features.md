
# Cangjie Extension Features

The Cangjie extension provides a comprehensive set of features to enhance your development experience with the Cangjie programming language.

## Syntax Highlighting

The extension provides comprehensive syntax highlighting for Cangjie code using a custom tree-sitter grammar, including:

- Keywords and operators
- Strings, numbers, and booleans
- Functions, classes, and interfaces
- Comments and documentation
- Control flow statements
- Pattern matching and enums
- Generics and type annotations

## Language Server Features

The extension integrates with the Cangjie Language Server to provide advanced code intelligence features:

- **Code Completion**: Smart suggestions for variables, functions, and types as you type
- **Go to Definition**: Jump to the definition of any symbol
- **Find References**: Find all references to a symbol in your codebase
- **Hover Documentation**: View documentation for symbols when hovering over them
- **Code Formatting**: Automatically format your code according to Cangjie style guidelines
- **Error Reporting**: Real-time error checking and diagnostics
- **Refactoring**: Rename symbols and extract code into functions
- **Code Navigation**: Jump between related symbols and files
- **Semantic Highlighting**: Enhanced syntax highlighting based on symbol meaning

## Debugging Support

The extension provides integrated debugging support for Cangjie programs:

- **Launch Configuration**: Debug your Cangjie programs with customizable launch configurations
- **Breakpoints**: Set breakpoints to pause execution at specific lines
- **Variable Inspection**: View and modify variables during debugging
- **Call Stack**: Navigate the call stack during debugging
- **Step Through Code**: Step through your code line by line
- **Expression Evaluation**: Evaluate expressions during debugging

## Task Templates

The extension includes pre-configured task templates to help you quickly run common commands:

- **Run Cangjie Program**: Execute the current Cangjie file
- **Build Cangjie Project**: Build the current Cangjie project
- **Test Cangjie Project**: Run tests for the current Cangjie project
- **Format Cangjie Code**: Format the current Cangjie file

## Snippets

The extension provides a set of useful code snippets to help you write code faster:

- `fn`: Create a new function
- `class`: Create a new class
- `interface`: Create a new interface
- `enum`: Create a new enum
- `if`: Create an if statement
- `match`: Create a match expression
- `for`: Create a for loop
- `while`: Create a while loop

## Theme Support

The extension includes a custom theme optimized for Cangjie code, with support for:

- Dark and light variants
- Syntax highlighting for all Cangjie language features
- Consistent colors across different symbol types
- Optimized for readability

## Integration with Other Tools

The extension integrates with other tools in the Cangjie ecosystem:

- **Cangjie Package Manager**: Support for managing dependencies
- **Cangjie Formatter**: Automatic code formatting
- **Cangjie Linter**: Static code analysis
- **Cangjie Debugger**: Advanced debugging capabilities

## Performance Optimization

The extension is optimized for performance, with:

- Efficient parsing using tree-sitter
- Background processing for non-blocking operation
- Memory-efficient design
- Support for large files
- Fast startup time
