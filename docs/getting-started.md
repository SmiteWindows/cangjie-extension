# Getting Started with Cangjie Extension

## Installation

1. Open Zed
2. Go to Extensions (Cmd+Shift+X on macOS, Ctrl+Shift+X on Windows/Linux)
3. Search for "Cangjie"
4. Click Install

## First Steps

1. Create a new file with `.cj` extension
2. Start writing Cangjie code
3. Enjoy syntax highlighting and language server features
